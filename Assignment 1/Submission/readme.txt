Libraries used:
	spaCy: https://spacy.io/usage
	nltk: https://www.nltk.org/install.html
	TextBlob: https://textblob.readthedocs.io/en/dev/install.html
	BeautifulSoup: https://www.crummy.com/software/BeautifulSoup/bs4/doc/#installing-beautiful-soup
	JobLib: https://joblib.readthedocs.io/en/latest/installing.html
Setup:
	1) Open the notebook in a Google Colab/Jupyter Notebook environment with the above libraries installed.
	2) Install the spaCy large model first and then restart the runtime to have it load correctly.
	3) Run the script that converts the provided json file to the correct format.
	4) Change the file path in the following cell to the current directory
	5) Run the rest of the Jupyter Notebook
	6) For the application, you can rerun the cell multiple times and write your own test review.
	7) Our saved Model for the Application component has been provided in the submission folder if you choose to use it.
Output:
	Jupyter Notebook has been annotated with comments about the code functions.
